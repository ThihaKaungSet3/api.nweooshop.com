
```POST "/api/register"```

This API is used to create new shop after user is registered.

Seller Params 

```
    {
	    "shopName" 	: <string>,
    	"city" 		: "<email>",
        "state"     : "<string>",
    }
```

Response Errors

```
    500 -> server error
```
Response Success

```
    200 	-> success 
```
